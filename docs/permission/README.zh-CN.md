## Permission 使用手册

[English](README.md) | 简体中文

本手册面向平台管理员/租户管理员，说明如何对 `permission` 子系统进行**初始化**与**日常管理**，包括创建**租户**、**组织**、**用户**、**角色**，以及配置**菜单/API**权限等。

### 系统功能介绍

- **多租户管理**：创建/维护租户信息，支持按域名访问（如启用租户域名识别）。
- **多应用集成**：支持在同一套权限体系下管理多个应用（应用列表、跳转与登录集成）。
- **OAuth2 单点登录**：提供统一登录与令牌签发能力，业务应用可通过 OAuth2 接入实现登录态与鉴权能力复用。
- **组织与用户**：维护组织树、用户账号与组织归属关系。
- **角色与授权（RBAC）**：为角色绑定菜单与 API 权限，并为用户分配角色。
- **数据域（数据范围）**：为角色配置数据访问范围（如本组织/仅本人/全租户等），用于业务数据的行级过滤。
- **多语言支持**：管理端/业务端可按需启用中英文等多语言切换。
- **运维能力**：登录/操作日志、白名单等辅助能力（详见相关章节）。
