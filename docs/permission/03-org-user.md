## 03. Organization & users

English | [简体中文](03-org-user.zh-CN.md)

### Goal

Build an organization structure and create users under the tenant.

### Steps (UI)

1. **Create organizations**
- Design an org tree (e.g. HQ / BU / Dept).
- Create nodes and verify the hierarchy.

2. **Create users**
- Create a user and set enabled/disabled state.
- Assign the user to an organization if applicable.

3. **Password operations**
- Reset user password (admin action).
- User changes password after login.

### Verification

- [ ] Org tree is displayed correctly
- [ ] Users list shows the new user
- [ ] Disabled users cannot log in
